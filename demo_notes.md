# Mocap Session Notes

> **Author:** Arshad Naguru · **Last Updated:** April 2026 · **Category:** How To


mocap session notes
- turn on optitrack cameras first
- open motive software on main pc
- place reflective markers on suit
- perform wand calibration

!!! note
    always calibrate before every new session


!!! warning
    never move cameras after calibration or you have to redo it

code: python start_session.py --mode record --fps 120
output files saved as .fbx in /sessions/output/
if markers drop out check camera exposure settings
