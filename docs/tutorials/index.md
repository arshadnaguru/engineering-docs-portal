# Tutorials

Tutorials are **learning-oriented** guides that take you through a series of steps to complete a project or learn a concept. They are designed for beginners who need hands-on experience.

## Available Tutorials

- [Setting Up a Python ML Environment](python-ml-setup.md) — Configure a reproducible Python environment for machine learning projects
- [Containerizing a FastAPI App with Docker](docker-fastapi.md) — Package and deploy a FastAPI application using Docker
- [Getting Started with Git for Documentation](git-for-docs.md) — Learn Git fundamentals in the context of documentation workflows

!!! tip "What makes a good tutorial?"
    A tutorial should guide the learner through completing a meaningful task from start to finish. The reader should have a working result at the end. Focus on **doing**, not explaining.
