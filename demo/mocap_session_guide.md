# Mocap Session Setup Guide

> **Author:** Arshad Naguru · **Last Updated:** April 2026 · **Category:** How To


- Turn on OptiTrack cameras from the power strip
- Open Motive software on the main workstation
- Place reflective markers on the suit at all joint positions
- Perform wand calibration using the calibration wand
- Set frame rate to 120fps in Motive settings
- Create a new session and name it with date and project name
- Ask the performer to stand in T-pose for 10 seconds
- Begin recording from the record button in Motive
- Review the capture for any marker dropouts before saving
- Export the session as FBX for Unreal Engine use

!!! note
    always calibrate before every new recording session even if cameras were not moved


!!! warning
    never reposition cameras after calibration or the entire calibration must be redone


!!! warning
    always save the raw Motive session file before exporting to FBX
